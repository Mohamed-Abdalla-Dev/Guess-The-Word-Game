# 🔤 Guess The Word Game

A simple word guessing game built with JavaScript.  
The player tries to guess the correct word based on given hints.

## 🚀 Features

- Guess the hidden word  
- Input validation  
- Simple feedback system  
- Interactive UI  

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript(Vanilla)  

## 📌 Goal

Practice JavaScript logic and working with user input.
